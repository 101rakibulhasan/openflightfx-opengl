# OpenFlightFX
OpenFlightFX is a dynamic plane game that generates terrain and are not limited to a certain area.

![OpenFlightFX](preview.png)

## How to play
G - Start game
M - Main menu
W, A, S, D - Up, left, down, right
z - Zoom out
Z - Zoom in
